# 🐍 Proyectos de Automatización en Python

## 📁 Descripción del repositorio

Este repositorio contiene **9 proyectos desarrollados en Python**, enfocados en la automatización de tareas del sistema y utilidades generales.

## ⚙️ Funcionalidades principales

Los scripts incluidos permiten realizar las siguientes tareas:

- 🗂️ Organización automática de carpetas  
- 🧹 Eliminación de archivos duplicados descargados  
- 📶 Obtención de claves WiFi guardadas en el sistema  
- ➕ Clase de operaciones matemáticas básicas  
- 🤖 Diversas automatizaciones para tareas del sistema  

## 🚀 Objetivo del proyecto

El objetivo de este repositorio es centralizar varias herramientas en Python que faciliten la automatización de tareas comunes, mejorando la eficiencia y el manejo del sistema.

## 🧠 Tecnologías utilizadas

- Python 3.x  
- Librerías estándar de Python (os, shutil, etc.)

## 📌 Notas

Este proyecto está en constante mejora e incluye scripts independientes que pueden ejecutarse por separado según la necesidad del usuario.
